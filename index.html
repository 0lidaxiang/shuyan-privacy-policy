我们处理哪些信息

🎤
麦克风录音 临时
你点击录音时，应用捕获语音音频并发送至语音识别接口。识别完成后，本地音频文件立即删除，不做任何保存。
📝
处理后的文本 仅本地
语音识别结果和 AI 整理后的文本，仅存储在你设备本地的数据库中，不上传至我们的服务器。
🔐
API 凭证 加密存储
你填写的豆包 API 凭证使用 iOS 系统钥匙串（Keychain）加密存储，只有本应用可以读取。
🚫
不收集的信息
我们不收集：姓名、邮箱、手机号、位置、设备标识符、联系人、相册或任何其他个人信息。应用无需注册账号。
2

信息如何被使用

应用对信息的使用完全服务于你发起的单次操作：

➊
语音识别
音频发送至豆包语音识别 API，获得文字结果后，音频文件随即删除。
➋
文本整理
识别出的文字发送至豆包大语言模型，经 AI 整理为结构化书面文本后返回给你。
➌
历史记录
整理结果存储在你设备的本地数据库，供你日后查看。你可以随时在 App 内删除全部历史记录。
3

第三方服务

书言使用字节跳动旗下的豆包 API 进行语音识别和文本处理。你的语音和文本数据将在处理过程中经过字节跳动的服务器。

🤖
豆包 API
字节跳动旗下火山引擎
提供语音识别（ASR）与大语言模型（LLM）服务。数据处理遵循字节跳动的隐私政策。字节跳动在处理过程中可能会保留必要的日志，具体以其服务协议为准。
查看火山引擎隐私政策 →

除此之外，应用不集成任何第三方分析、广告或数据收集 SDK。

4

数据安全

🔒
传输加密
与豆包 API 的所有通信均使用 TLS 1.2+ 加密传输。
🗝️
凭证保护
API 凭证存储于 iOS Keychain，由系统硬件安全模块保护，不写入任何明文文件。
📦
本地隔离
历史记录数据库存储在应用沙盒内，其他应用无法访问。我们也无远程访问该数据的能力。
5

你的权利与选择

🗑️
删除历史记录
在 App 内「历史记录」页面，可随时删除单条或全部历史记录。
🎙️
撤销麦克风权限
前往「设置 → 书言」，可随时关闭麦克风权限。关闭后录音功能将不可用。
💾
完全卸载
卸载应用将同时删除本地所有数据，包括历史记录和 Keychain 中的凭证。
